# 项目名称
Stardew_Valley_Farm
# 项目简介
该项目是一个基于 [cocos2d-x 3.17.2](https://docs.cocos.com/cocos2d-x/manual/)开发的星露谷农村游戏。
该游戏是一个农场生活模拟游戏，玩家将管理自己的农场，种植作物，养殖动物，与镇上的居民互动，参与节日活动，并探索周围的自然环境。游戏应包含丰富的季节变化、社区交互和个人角色发展。
